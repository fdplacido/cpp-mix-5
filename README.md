## C++ MIX #5

## Requirements

- CMake
- Python 2.7
- C++14 compiler

### Official supported platforms
#### macOS

```
$ brew install cmake
```

#### Debian 9 Stretch

```
$ sudo apt-get update
$ sudo apt-get install git cmake g++ python libfontconfig1-dev \
    libx11-dev libxcomposite-dev libgl1-mesa-dev libglu1-mesa-dev freeglut3-dev -y
```

## Getting started

```
$ git submodule update --init
$ cmake .
$ make

$ ./bin/example
```
